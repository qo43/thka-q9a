# Wathiq Prototype (OCR)

## How to Run
1. Open the project in Intellij.
2. Run 'WathiqApplication.java'.
3. The Backend runs on: 'http://localhost:8080'.

## API Endpoints
- **POST** '/api/scan': Upload a file ('form-data' key: 'file'). Returns JSON.

## Test Files
Just run the application and go to the localhost a html page should be loaded
then upload any image (it should be an image) containing text and the output
will be printed on the page in JSON format.
